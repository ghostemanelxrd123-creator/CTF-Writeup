## Task description
Снят дамп bluetooth трафика с устройства. Найдите флаг. Подсказка: обратите внимание на obex пакеты

## Analysis
В результате первичного анализа obex трафика, я обнаружил put пакет, содержащий большое количество закодированных данных, для восстановления исходного содержимого
был проведен ряд тестов с использованием различных методов декодирования: Пробовал раскодировать в разных системах кодировки, попробовал utf-8, hex, base64. Ничего не получалось.
Но при интерпретации данных как ASCII последовательности, в одной из вариаций мне удалось извлечь исходный текст. Среди извлеченных строк мной был найден флаг, соответсвтующий формату.
