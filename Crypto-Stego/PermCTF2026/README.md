## Task description
Анализ массива данных, нахождение скрытых записей, извлечение флага.

## Analysis
В ходе первичного анализа дампа, мной были выявлены следующие аномалии:
1. **Формат номера карты:** В некоторых строках номера содержали шестнадцатиричные символы, что недопустимо для реальных банковских карт.
2. **Валидация дат:** Срок действия в тех же строках не соответствовал настоящим датам, например, 05/96.
3. **Паттерн CVV:** Аномальные строки сопровождались CVV состоящими из последовательных цифр.

## Solution
Совокупность всех этих аномалий в строках дала наводку, что именно эти строки могут содержать искомую информацию.
Перепробовав разные методы извлечения данных из номеров карт аномальных строчек, такие как декодировка base64, ASCII, извлечь флаг мне не удалось.
Тогда я попробовал перевести эти номера из шестнадцатиричной системы счисления в текст.
Как оказалось, эти номера были частями от полного текстового флага.

**CVV 111**: `707374756374667B` => `pstuctf{`

**CVV 222**: `68656C6C6F5F6974` => `hello_it`

**CVV 333**: `5F6C756E685F616C` => `_lunh_al`

**CVV 444**: `676F726974686D7D` => `gorithm}`

## Flag
И так, подставив декодированные значения я получил полный флаг:
```
**pstuctf{hello_it_lunh_algorithm}**
```
